# 智能门禁安防传感器节点 — 接线说明文档

> **项目**: 智能门禁访客管理系统 — Sensor-C 安防检测节点
> **主控芯片**: TI CC2530 (ZigBee SoC, 8051内核, 3.3V逻辑电平)
> **文档版本**: v1.0
> **日期**: 2026-06-08
> **作者**: Lixm

---

## 1. 硬件概述

Sensor-C 节点负责门禁系统的安防检测功能，集成以下传感器：

| 编号 | 传感器类型 | 模块型号 | 检测功能 | 输出类型 |
|:---:|----------|---------|---------|:---:|
| ① | PIR 人体红外 | HC-SR501 / AM312 | 访客靠近检测 | 数字 (高有效) |
| ② | REED 霍尔门磁 | KY-003 / AH3144 | 房门开关检测 | 数字 (低=闭合) |
| ③ | 震动传感器 | SW-420 / KY-002 | 敲门/破坏检测 | 数字 (低=震动) |
| ④ | TCH 触摸按键 | TTP223 / 电容触摸 | 门铃触发检测 | 数字 (低=触摸) |

## 3. 各传感器接线详图

### 3.1 PIR 人体红外传感器 (HC-SR501)

```
     HC-SR501 模块                           CC2530
  ┌────────────────┐                   ┌──────────────┐
  │                │                   │              │
  │  VCC  ──────────────────────────── 3.3V          │
  │  GND  ──────────────────────────── GND           │
  │  OUT  ──────────────────────────── P0.0          │
  │                │                   │              │
  └────────────────┘                   └──────────────┘

  模块配置 (通过背面跳线/电位器):
  ├─ 触发模式: 单次触发 (跳线帽插在 L 位置) 或 重复触发 (H 位置)
  ├─ 延时时间: 逆时针调至最小 (~2s) — 由 CC2530 软件去抖
  └─ 灵敏度:   调至中间位置 (~5m 检测距离)

  ⚡ 电气参数:
  ├─ 工作电压: DC 5V~20V (模块内置 LDO, 可接 CC2530 的 3.3V)
  ├─ 静态电流: < 50μA
  ├─ 输出电平: 高 3.3V / 低 0V
  └─ 检测角度: 120° 锥形

  📐 安装建议:
  ├─ 安装高度: 2.0~2.5m (门框上方)
  ├─ 检测方向: 朝向门口外侧
  └─ 避开: 空调出风口、阳光直射、热源
```

### 3.2 REED 霍尔门磁传感器 (AH3144 / KY-003)

```
      霍尔模块 (门扇侧)                      磁铁 (门框侧)
  ┌──────────────────┐                   ┌──────────┐
  │  VCC ────── 3.3V │                   │          │
  │  GND ────── GND  │                   │   N      │
  │  OUT ────── P0.2  │                   │   S      │
  │                  │                   │          │
  │  [霍尔芯片]       │ ←── 磁场 ──────── │  [永磁铁] │
  └──────────────────┘                   └──────────┘
      门扇上                             门框上

  ⚡ 电气参数:
  ├─ 工作电压: DC 3.3V~5V
  ├─ 输出: 有磁场 → 低电平 (DOOR_CLOSED)
  │        无磁场 → 高电平 (DOOR_OPEN)
  └─ 检测距离: < 1cm (磁铁与霍尔间距)

  📐 安装建议:
  ├─ 霍尔模块固定在门扇上边缘
  ├─ 磁铁固定在门框对应位置
  ├─ 门关闭时霍尔与磁铁对齐且间距 < 8mm
  └─ 如需常闭/常开反转，修改 hall.h 中 DOOR_STATE 定义
```

### 3.3 震动传感器 (SW-420 / KY-002)

```
     震动模块                                CC2530
  ┌────────────────┐                   ┌──────────────┐
  │                │                   │              │
  │  VCC  ──────────────────────────── 3.3V          │
  │  GND  ──────────────────────────── GND           │
  │  DO   ──────────────────────────── P0.1          │
  │                │                   │              │
  │  [电位器]       │                   │              │
  └────────────────┘                   └──────────────┘

  ⚡ 电气参数:
  ├─ 工作电压: DC 3.3V~5V
  ├─ 静态输出: 高电平 (不震动)
  ├─ 触发输出: 低电平脉冲 (震动触发)
  └─ 灵敏度: 通过模块电位器调节

  📐 安装建议:
  ├─ 安装在门扇内侧 (检测敲门和破坏)
  ├─ 确保模块与门体刚性连接
  ├─ 灵敏度调节: 轻敲门可触发，关门震动不误触发
  └─ 可用作: 门铃触发 / 破坏检测 / 地震检测
```

### 3.4 TCH 触摸按键 (TTP223 / 电容触摸模块)

```
     触摸模块                                CC2530
  ┌────────────────┐                   ┌──────────────┐
  │                │                   │              │
  │  VCC  ──────────────────────────── 3.3V          │
  │  GND  ──────────────────────────── GND           │
  │  OUT  ──────────────────────────── P0.0 (复用)   │
  │                │                   │              │
  │  [触摸区域]     │                   │              │
  └────────────────┘                   └──────────────┘

  ⚡ 电气参数:
  ├─ 工作电压: DC 3.3V~5V
  ├─ 工作模式: 点动 (触摸输出低, 松开恢复高)
  ├─ 响应时间: < 60ms
  └─ 触摸介质: 可穿透 3mm 亚克力/玻璃

  📐 安装建议:
  ├─ 门铃按钮位置 (门外侧, 1.3m 高度)
  ├─ 触摸面板需与外壳绝缘
  ├─ 引线使用屏蔽线 (长度 < 30cm) 防止误触发
  └─ ⚠️ Mode=2 时才使能触摸驱动 (P0.0 复用)
```

## 5. 电源接线

```
   USB 5V / 电池                  CC2530 + 传感器
  ┌──────────────┐              ┌─────────────────┐
  │              │              │                 │
  │  5V  ────────────────────── VIN (板载LDO)     │
  │  GND ────────────────────── GND              │
  │              │              │                 │
  │              │              │  3.3V ───┬── PIR VCC    │
  │              │              │         ├── 霍尔 VCC    │
  │              │              │         ├── 震动 VCC    │
  │              │              │         └── 触摸 VCC    │
  └──────────────┘              └─────────────────┘

  ⚡ 供电要求:
  ├─ CC2530 工作电压: 3.3V (绝对最大 3.9V!)
  ├─ 总电流估算: CC2530 ~30mA + 传感器 ~20mA = ~50mA
  ├─ 推荐使用: 5V/1A USB 适配器 + 板载 AMS1117-3.3
  ├─ 电池供电: 3.7V 锂电池 + HT7333 LDO
  └─ 所有传感器供电共用 CC2530 的 3.3V 输出
```

## 6. 接线检查清单

| 检查项 | 标准 | ✓ |
|--------|------|:---:|
| 所有 VCC 接 3.3V | 万用表测 VCC-GND = 3.30 ±0.15V | |
| 所有 GND 共地 | CC2530 GND ↔ 传感器 GND 导通 | |
| PIR OUT → P0.0 | 无虚焊、无短路 | |
| 霍尔 OUT → P0.2 | 门关闭时 P0.2 = 0V, 门打开 = 3.3V | |
| 震动 DO → P0.1 | 静止 P0.1 = 3.3V, 敲击 = 低脉冲 | |
| 触摸 OUT → P0.0 | 仅 Mode=2 时连接 | |
| 杜邦线连接可靠 | 轻拉不脱落 | |
| 无短路 | 上电前测 VCC-GND 电阻 > 1kΩ | |

## 7. 注意事项

1. **电平匹配**: CC2530 是 3.3V 系统，所有传感器必须使用 3.3V 版本或加电平转换
2. **P0.0 复用**: PIR 和触摸传感器共用 P0.0，同一时刻只能接一个 (由 Mode 参数切换)
3. **中断配置**: 所有传感器输入引脚已配置为下降沿中断 (PICTL P0ICON=1)
4. **上拉电阻**: 传感器 OUT 引脚需外接 10kΩ 上拉电阻到 3.3V (部分模块已内置)
5. **去耦电容**: 建议在各传感器 VCC-GND 间并联 0.1μF 电容
6. **ESD 防护**: 安装时佩戴防静电手环，避免损坏 CC2530 GPIO
7. **线缆长度**: 传感器信号线建议 < 50cm，过长需加屏蔽
8. **测试工具**: 准备万用表 + 逻辑分析仪 (或示波器) 用于信号调试
